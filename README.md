## Overview

OS-NVR is a mobile-friendly extensible CCTV system.

Use [Issues](https://codeberg.org/Curid/os-nvr/issues) for bug reports, feature requests and support. [Matrix chat](https://matrix.to/#/#os_nvr:matrix.org)
